# Система управления банковскими картами

Описать данный файл по ТЗ

 - Скопировать репозиторий командой git clone https://github.com/maximl93/bank-rest-app.git
 - Перейти в корневую папку проекта.
 - Запустить команду docker-compose up --build. Команда упакует приложение и запустит контейнеры с БД и приложением.
 - После запуска приложения в браузере перейти по ссылке http://localhost:8080/swagger-ui/index.html#/
